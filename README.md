# Personal Portfolio Website – Task 1

## Overview
This project is a multi-page personal portfolio website developed as part of **Thiranex Task 1: HTML5 Semantic Structure & Accessibility**. The website is built using semantic HTML5 elements and follows accessibility and SEO best practices.

## Features
- Semantic HTML5 structure
- Multi-page website
- Navigation menu
- About page
- Projects page
- Contact page
- Accessible contact form
- ARIA labels for accessibility
- SEO meta tags
- Responsive HTML structure

## Pages
- Home (index.html)
- About (about.html)
- Projects (projects.html)
- Contact (contact.html)

## Technologies Used
- HTML5
